# Helpy Local Backend

Adds AiderDesk status UI and commands for a local llama.cpp Docker Compose backend.

```text
http://127.0.0.1:8080/v1
/home/shingen/Tech Projects/Helpy/docker-compose.yml
```

Commands:

```text
/helpy-backend-status
/helpy-backend-start
/helpy-backend-stop
/helpy-backend-logs
```
